Seo y comentarios en HTML

El seo (search engine optimization) de una pagina web es importante si quieres que la pagina web resalte en el buscador del navegador, en el apartado del html podemos ayudar a que el nivel de seo aumente de la siguiente manera

html lang="en":

Ayuda al navegador a identificar mas rapido el lenguaje de nuestra pagina

meta charset="UTF-8":

Declarar el decodificador que se utilizara

meta name="viewport" content="width=device-width, initial-scale=1.0":

Ayuda a que los dispositivos moviles empiecen con un zoom predeterminado y no se vea tan peque√±o el contenido

link rel="icon" href="../../assets/images/Yo.png" type="image/png":

Puedes definir el icono de la pagina

meta name="description" content="Pagina web que prueba mis conocimientos en HTML y CSS":

Asigna una descripcion del sitio para aumentar el nivel del SEO

meta name="keywords" content="Aprender HTML, Aprender CSS, HTML, CSS":

Asigna palabras clave para tener mas probabilidad de aparecer en busquedas

meta name="author" content="Carlos Guardiola":

Asigna el author del sitio

meta name="title" content="Aprende HTML y CSS desde Cero":

Asigna el titulo de la pagina (muestra parte superior)

meta name="robots" content="":

Configura si requieres que el sitio aparezca o no en el buscador

Formularios avanzados

En este apartado conoceremos 5 elementos html que nos permiten realizar formularios mas avanzados.

  • textarea: Permite ingresar al usuario multiples lineas de texto, y adecuar su campo conforme se va escribiendo.
  • select: Permite al usuario desplegar una lista con valores predeterminados para seleccionar
  • datalist: Permite al usuario ingresar texto y desplegar una lista con autocompletado
  • fieldset & legend: Permite seccionar un formulario y definirle un nombre

Textarea

  • disable:: Desactiva el campo y evita el envio de informacion
  • read-only: Activa el campo, pero no puede editar su contenido
  • max-length: Asigna un limite para ingresar caracteres

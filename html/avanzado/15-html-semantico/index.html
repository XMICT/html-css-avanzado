HTML Semantico

Una buena estura semantica en HTML permite al navegador posicionar una pagina web en el buscador, ademas de transformarla en una pagina inclusiva para que cualquier tipo de usuario tenga acceso a ella.

Buenas practicas para que tu pagina tenga una buena estructura HTML

  • Agregar el atributo alt a los elementos multimedia
  • Agregar el atributo aria-label a los elementos

    • El aria-label, permite describir al elemento

  • Utilizar correctamente cada etiqueta
    • header
    • nav
    • main
    • aside
    • footer

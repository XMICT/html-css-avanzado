Etiquetas importantes

Algunas etiquetas con el tiempo fueron quedando obsoletas, sin embargo algunas otras llegaron para establecer una mejor semantica en la pagina y aumentar el SEO en la misma, algunas de estas son:

  • b -> strong: Importante

    • Sirve para marcar un texto como importante en la pagina, automaticamente convierte el texto en negritas.

  • i -> em: Enfasis

    • Sirve para dar enfasis a un texto

  • strike -> del, s: Tachado

    • Efecto de tachado en un texto

  • tt -> code: console.log('Hola mundo')

    • Sirve para ingresar codigo

  • u -> span con estilo: Subrayado

    • Subrayar un texto

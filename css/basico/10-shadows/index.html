Shadows

Las sombras son efectos en CSS que hacen que los elementos sobresalgan de un fondo, existen 3 tipos de sombras

  • box-shadow: El box shadow es utilizado para implementar una sombra en un contenedor
  • text-shadow: El text shadow es utilizado para implementar una sombra en un texto
  • drop-shadow: El drop shadow es utilizado para implementar una sombra en un elemento

Ejemplo

  • box-shadow: 0 0 8px 0 #0002 -> offsetX, offsetY, radio, difuminacion
  • text-shadow: 1px 1px 3px #0073ff -> offsetX, offsetY, radio
  • drop-shadow: 0px 0px 10px #000a -> offsetX, offsetY, radio

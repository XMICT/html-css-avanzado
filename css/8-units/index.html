Medidas en CSS

Las medidas en CSS pueden derivarse de dos formas

  • Absolutas: px, mm, cm, in (Estas ultimas 3 son en caso que la pagina tenga que imprimirse)
  • Relativas
    • Cajas
      • Porcentaje: Si se define un porcentaje es porque la mediada sera respecto al contenedor padre
      • vw: Viewport width definira el tamaño del elemento respecto al ancho total de la pantalla
      • vh: Viewport height definira el tamaño del elemento respecto al alto total de la pantalla
    • Texto:
      • em: Se utiliza cuando se define un tamaño en el contenedor como es la fuente 12px y si se requiere que un elemento hijo tenga la mitad de este tamaño se declara como 0.5 em = 6px
      • rem: Rem funciona casi igual al em, solo que rem se ubica en la etiqueta mas alta de la pagina (html)

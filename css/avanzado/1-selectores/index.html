Selectores

Los selectores son la forma de seleccionar elementos en el DOM para aplicarles estilos.

Ejemplos

Selector por atributo

Selecciona todos los elementos que tengan el atributo yo.png

Ejemplo: 

img[src$=".png"] {
    border: 1px solid red;
}

Selectores para hijos

Selecciona todos los elementos que sean hijos y nietos de un elemento

El section tendra un estilo para que todos sus hijos y nietos li tengan el mismo color. Pero si uno de estos elementos se encasilla en un div y se le asigna un selector con hijo directo, este ahora sera el nuevo color.

  • Elemento nieto de section
  • Elemento hijo directo de section

    • Ejemplo: 

    section p{
    color: #f00;
}

div > p{
    color: #0f0;
}

    Selectores adyacentes cercano al siguiente

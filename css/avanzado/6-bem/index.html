Metodologia BEM

Es una metodologia que permite estructurar el codigo de CSS de una manera mas ordenada y escalable, permitiendo que el codigo sea mas facil de mantener y de entender.

  • Bloques: Elemento independiente, que puede insertarse en cualquier parte
  • Elementos: Elemento dependiente del bloque, un conjunto de estos permite darle sentido al mismo
  • Modificadores: Permiten modificar el codigo

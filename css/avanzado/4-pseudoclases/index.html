Pseudoclases

Las pseudoclases son palabras clave para estilizar el estado de los elementos. Ejemplos de estos estados pueden ser:

  • hover: Cuando el mouse esta encima de elemento
  • active: Cuando hay un evento click en el elemento
  • first-child/last-child: Selecciona el primer y el ultimo elemento de un conjunto de elementos hermanos
  • nth-child(x): Selecciona el elemento x de los hermanos
  • not(x): excluye el elemento x
  • empty: selecciona elementos vacios
  • root: Selecciona el elemento raiz
  • checked: Verifica si el elemento esta checado
  • link: Funciona para darle color a los enlaces no visitados
  • visited Funciona para darle color a los enlaces visitados
  • invalid/valid: Es util para inputs y estos modifican el estilo si el estado es valido

  • section has(.element): Modifica el elemento padre si contiene element
  • is(.element1, element2, element3): Agrupa los elementos y modifica la especificidad con un nivel mas arrib
  • where(.element1, element2, element3)Agrupa los elementos

Active y mouseover

Estado de un boton, active para hacer click y mouseover para pasar el cursor por encima

Ejemplos

Estado actual: 

#btn-state{
    padding: 12px 16px;
    border-radius: 8px;
    border: none;
    background-color: #8f0000;
    color: white;
    font-weight: bold;
    transition: all 0.2s ease-in-out;
    text-transform: uppercase;
}

#btn-state:active{
    background-color: #008f00;
    transition: all 0.2s ease-in-out;
}

#btn-state:hover{
    background-color: #00008f;
    transition: all 0.2s ease-in-out;
}

Object fit

La propiedad object fit es parecida al background, con la diferencia que esta aplica para elementos con la etiqueta imagen

  • object-fit:
    • contain: Aleja la imagen para que pueda visualizarse toda
    • cover: Hace zoom en la imagen para que abarque tanto el width y height
    • none: se basa de la resolucion de la imagen original para mostrarse
    • scale-down: Compara el tamaño de la imagen original con la declarada en el css y se queda con la medida mas pequeña

Ejemplo

object-fit: contain

object-fit: cover;

object-fit: none;

object-fit: cover; + object-position: right;

Herencia

La herencia es el traspaso de propiedades a un elemento hijo, en un elemento se puede especificar si se necesita utilizar o no herencia, el valor por defecto para los atributos inherit

Posibles propiedades para asignar a los atributos

  • inherit: Tomara el valor heredado de cualquier elemento padre
  • initial: Tomara el valor inicial o el valor por defecto del navegador
  • unset: Restablece el su valor a inicial si no se encuentra la propiedad en el padre directo
  • revert: Busca los estilos del navegador

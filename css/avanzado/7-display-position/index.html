Display y position

Display y position son propiedades que nos permiten controlar la forma en la que los elementos se muestran en la página.

Display

La propiedad display nos permite controlar la forma en la que los elementos se muestran en la página. Algunos valores de display son:

  • block: Hace que el elemento se muestre como un bloque, ocupando todo el ancho disponible.
  • inline: Hace que el elemento se muestre en línea, ocupando solo el espacio que necesita.
  • inline-block: Hace que el elemento se muestre en línea, pero permite aplicar márgenes y padding.
  • flex: Hace que el elemento se muestre como un contenedor flexible.
  • grid: Hace que el elemento se muestre como un contenedor de cuadrícula.

Position

La propiedad position nos permite controlar la posición de los elementos en la página. Algunos valores de position son:

  • static: Hace que el elemento se muestre en la posición en la que aparece en el flujo del documento.
  • relative: Hace que el elemento se muestre en la posición en la que aparece en el flujo del documento, pero permite moverlo con las propiedades top, right, bottom y left.
  • absolute: Hace que el elemento se muestre en la posición en la que aparece en el flujo del documento, pero permite moverlo con las propiedades top, right, bottom y left, tomando como referencia el elemento padre más cercano con una posición diferente a static.
  • fixed: Hace que el elemento se muestre en la posición en la que aparece en el flujo del documento, pero permite moverlo con las propiedades top, right, bottom y left, tomando como referencia la ventana del navegador.
  • sticky: Hace que el elemento se muestre en la posición en la que aparece en el flujo del documento, pero permite moverlo con las propiedades top, right, bottom y left, tomando como referencia el elemento padre más cercano con una posición diferente a static, y se queda fijo cuando se hace scroll.

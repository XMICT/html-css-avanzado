Herencia, Cascada y Especificidad

Los siguientes conceptos son importantes a la hora de asignar propiedades a los elementos, ya que si no seguimos de forma adecuada las normas puede que tengamos resultados no esperados, por ello es importante entenderlos

  • Herencia: La herencia no es mas que el traspaso de propiedades de un elemento padre a un elemento hijo
  • Cascada: La cascada es la manera en la que trabaja css (cascade style sheet), quiere decir que las propiedades de los elementos se asignaran prioritariamente de la ultima linea hacia la primera en el codigo CSS
  • Especificidad: Es el nivel de prioridad para asignar una propiedad. Este se conforma de tres valores y se ve de la siguiente forma -> (0,0,1). Mientras mas a la izquierda este el uno, es mas importante aplicar su atributo
